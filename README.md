## 杂记

- 一个身份证只能作为5个管理员
- 一个人开发者可以绑定50个小程序(扎心，目前好像已经绑定了快40个了)
- 解除绑定: 关注公众平台安全助手（mphelper），点击菜单-->绑定查询->微信号绑定帐号，可以看到绑了哪些小程序，然后也可以直接进行解绑。
